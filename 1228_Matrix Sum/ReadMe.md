# Matrix Sum

## TLE的想法

暴力，先存储从 `mat[0][0]` 到每个矩阵元 `mat[i][j]` 对应的矩阵元素和的奇偶性（bool）。
然后在遍历所有矩阵 `sum[x2][y2] ^ sum[x1][y2] ^ sum[x2][y1] ^ sum[x1][y1]` 计数即可。  
但是还是需要四重循环，30%点 TLE。  
做的时候发现一个小结论：n阶方阵子矩阵的数目是 $(\dfrac{n(n+1)}{2})^2$，所以只用算奇数偶数中的一个，另一个用总数减就可以了。
