# 十年

## 排序/优先队列

先求出 n^2 个数，然后排序。  
或者使用优先队列。  
时间复杂度O(n^2 logn)

## 优先队列-小根堆

优先队列存下标，并且维护一个表示第 i 行（即取 A[i] 时）当前 B 数组最小元素的数组 li[i]  
先将下标 1-n insert 进堆，然后每次出堆（输出和）下标 i 对应 li[i]++ ，再将 i insert 进堆。

## 进一步优化

可以考虑保存二元组 (x,y) 记录当前最小和位置然后向右向下遍历

## 注意！！！

按位或 `|` 优先级小于比较运算符 `>,<,>=,<=` 优先级！！！
所以 `tmp<<1|1<top` 会判断错误！！！
